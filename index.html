Progressive Enhancement

Progressive Enhancement allows the web developer to make a full interactive website by using a very simple key concept. This concept allows the developer to efficiently work on the website from the ground up. The analogy of Peanut m&m, the concept of working from inside out. The peanuts represent the strong base of all html files, providing a strong base foundation for extra content. The chocolate on top of peanut provides the foundation for the css to be applied and experimented to the developers desire. The wrapper for the m&m represents the endless possibilities of javascript. This will allow the developer to display and represent the website to their own desire with animations. Prgressive Enhancement is a concept of adding to your website one layer at a time.
